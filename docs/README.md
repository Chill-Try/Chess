## 本次会话变更索引

本次会话补充了几类重要能力，详细设计见 [ARCHITECTURE.md](./ARCHITECTURE.md)：

- 顶部栏增强：应用 Logo、浏览器页签 Logo、`残局对抗` 入口、作弊选项与电脑速度菜单
- 身份信息牌增强：轮到玩家时显示 `> 该你走棋了 <`，思考中提示支持呼吸动画；将死时显示 `获胜！`，和棋时双方显示 `和棋`
- 延迟动作调度：`重新开始`、作弊按钮、难度切换、`残局对抗` 都统一走“禁用按钮 → 取消后续电脑走棋 → 等待当前走棋完成 → 1 秒后生效”的流水线
- 升变交互：玩家兵升变时弹出置顶选项条，按“后、车、象、马”顺序提供手动选择
- 作弊规则：当前轮到谁走棋，就直接改写该方盘面，不校验标准合法性；对局结束后作弊按钮失效
- 音效更新：作弊复用胜利音效；胜利 / 失败 / 和棋分离；吃子音效改为更积极的单一版本
- Stockfish 难度增强：困难 / 大师使用动态深度；大师增加少量随机性与开局库支持；界面中的“当前深度”仅显示 `实际深度 - 4`
- 轻量残局识别：为 `hard` / `master` 识别若干基础必胜残局，并在命中时把 Stockfish 搜索深度直接提升到 `18`；同时支持从已接入模板随机生成残局练习局面

如果项目结构有调整，请优先同步更新这两份文档以及根目录的 [README.md](../README.md)。
